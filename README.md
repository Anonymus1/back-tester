# back-tester

A Rust-based backtesting application.

## Prerequisites

- Rust (1.70 or later)
- Cargo (comes with Rust)

## Building

To build the project:

```bash
cargo build
```

For a release build:

```bash
cargo build --release
```

## Running

To run the application:

```bash
cargo run
```

## Testing

To run tests:

```bash
cargo test
```

## Development

The project uses Cargo as the build system and package manager. The main source code is located in `src/main.rs`.
